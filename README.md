# java-lotto-precourse

## 기능 목록
1. 입력 및 검증
   - [ ] 로또 구입 금액 입력
      - [ ] 입력이 null, not int, 0, not 1000원 단위이면 예외 발생 후 재입력 시퀀스
   - [ ] 당첨 번호 입력(쉼표 기준 구분)
      - [ ] 각 상황에 맞는 입력 문구 문자열 출력
      - [ ] 쉼표 기준으로 파싱
      - [ ] 각 파싱된 항목이 1~45 범위의 정수인지 검증
      - [ ] 중복 여부 검증
      - [ ] 잘못된 입력 시 예외 발생 후 재입력 시퀀스
   - [ ] 보너스 번호 입력
      - [ ] 1~45 의 정수인지 검증
      - [ ] 당첨 번호와 중복 검증
      - [ ] 잘못된 입력 시 예외 발생 후 재입력 시퀀스
2. 로또 발행 및 표시
   - [ ] 장수 계산 (장수 = 구입 금액 / 1000)
   - [ ] 장수 만큼 로또 발행
      - [ ] 로또 번호 추첨
      - [ ] 발행된 각 로또 출력(숫자는 오름차순 정렬, 구분자는 `", "`)
3. 당첨 통계 및 수익룰 계산
   - [ ] 로또 번호와 당첨 번호 비교
      - [ ] 일치 개수 계산
      - [ ] 5개 일치인 경우 나머지 한 숫자와 보너스 숫자 일치 여부 확인
   -[ ] 등수별 당첨 개수 집계
   -[ ] 수익률 계산 (총 당첨금 / 총 구매금액 * 100, 소수점 두 자리까지)
4. 예외 처리
   - [ ] 잘못된 값 입력 핸들링 (아래 항목의 순서로 진행)
      - [ ] `IllegalArgumentException` 발생
      - [ ] "[ERROR]" 로 시작하는 해당 상황에 맞는 에러 메시지 출력
      - [ ] 해당 부분부터 입력 재개
      - [ ] Null 값 전달시 `NullPointerException` 발생 등 에러를 명확한 유형으로 처리
